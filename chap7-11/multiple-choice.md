# Multiple Choice

1. A polymorphic function (d) may resolve to
   values of different types, depending on inputs.
2. The composed function g . f has the type
   (b) Char -> [String]
3. (b) Num -> Num -> Bool
4. A function with the type (a -> b_) -> c
   (b) is a higher-order function
5. (a) f True :: Bool
